# Functions

## Function as a Relation

Note that since each function is a relation,
each theorem which applies to relations also applies to a function.

## Evenness

A function cannot be even if the domain is not "symmetric": (5, 5], for example.

## One-to-one functions

A horizontal line test can be used to determine whether a function is one-to-one or not.
Consider a semicircle, parabola, or a line, for example.

## Inverse functions

One has to prove that an inverse function does indeed exist.
If an inverse exists, it is symmetrical across `y = x` to the original function.
