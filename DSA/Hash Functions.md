# Hash Functions

## Array-based associations

One can create a simple associative mapping based on array
if the key space is of limited size - just use array indices as keys.

For example, validating whether two strings are anagrams or not
is a perfect example of a limited key space size of 26 characters:

https://neetcode.io/problems/is-anagram/question?list=neetcode150

Observe that the input string has to be lowercased to lie within this 26 character range.
