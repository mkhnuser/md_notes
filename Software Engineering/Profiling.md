# Profiling

## Timing

`time` utility allows you to examine how much time program takes to execute.
`hyperfine` utility allows you to reliably compare executing time of multiple programs.

## Performance Analysis

`perf` utility allows you to examine hotspots of your program.

Flame Graphs are convenient tools for performance visualization.
